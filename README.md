# week1-expense-splitter
Expense Splitter is a small full stack web application where users can add their expenses and track how much money each person spent. This project helps to calculate how much amount each person owes and how much balance is there between friends after spending. 



Expense Splitter (Week 1 Project)

This project is a small full stack web application where we can add, view and manage shared expenses between friends. When multiple people spend money in a group (trips, food, shopping etcâ€¦), it becomes confusing to find who owes how much amount. Expense Splitter helps to store all expenses in one place and later calculate balance between members. Our main purpose of this project is to practice team based development using Git & GitHub workflow with pull requests, reviews and branch protection rules.

Tech Stack

Frontend: React JS or (HTML,CSS,Javascript)

Backend: Node JS + Express

Database: MongoDB (Atlas) or(you can choose any database).

Features:

Add a new expense (title, amount, paidBy, date)

View all recorded expenses

Delete an expense

Simple summary view (later this week).
